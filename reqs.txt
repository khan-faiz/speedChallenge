## opencv
http://www.pyimagesearch.com/2016/10/24/ubuntu-16-04-how-to-install-opencv/
sudo apt-get install build-essential cmake pkg-config libjpeg8-dev libtiff5-dev libjasper-dev libpng12-dev libopencv-dev libgstreamer0.10-dev libgstreamer1.0-dev libgstreamer-plugins-base0.10-dev libgstreamer-plugins-bad1.0-dev  libgstreamer-plugins-good1.0-dev libgstreamer-plugins-base1.0-dev libgtk-3-dev python2.7-dev python3.5-dev
wget https://github.com/opencv/opencv/archive/3.2.0.zip
sudo apt-get install unzip
cmake . -DWITH_IPP=OFF
sudo make install
